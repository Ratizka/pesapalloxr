## Pesapalloxr

Pesäpalloxr on pesäpalloon luotu kerääjä kotiutustilanteita varten. Käyttäjä voi merkata lyönnnin, jonka avulla
lasketaan todennäköisyys juoksulle ja kuinka todennäköisesti se menee läpi. Riippuen kysynnästä ja ajasta kerääjään
voidaan lisätä muut tilanteet.

### Windows

Lataa pesapalloxr.exe ja suorita.

### Muut

Koitan tehdä asennuspakettia jokaiselle käyttöjärjestelmälle Windows, Mac ja Linux-pohjaiselle. 

Muut käyttöjärjestelmät voivat katsoa [linkki](https://github.com/Ratizka/pesapalloxr#käynnistys-suoraan-lähdekoodista)

### Käynnistys suoraan lähdekoodista

Vaatimukset Java 21 ja javafx.

``
git clone https://github.com/Ratizka/pesapalloxr.git

cd pesapalloxr

mvn clean javafx:run
